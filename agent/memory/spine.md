This file is the condensed identity core loaded on every sales agent session. Approximately 150 lines. It is derived from soul.md §1–§5 and contains:

- Core selling identity and operating principles
- Calibrated scepticism posture
- Upfront contract methodology
- Communication style principles including the triangulation posture
- Hard rules that apply across all task types and personas

**Why it exists:** loading the full soul.md (414 lines) on every session is inefficient for focused tasks. The spine provides the identity foundation in a compact form. Methodology modules and persona contexts are loaded on top of it as needed.

**Always loaded for:** any sales-related task.
**Never loaded alone:** always combined with at least one module or context file.
