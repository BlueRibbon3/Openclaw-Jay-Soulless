# Agent Memory — Expansion Plays and Sub-Vertical Packs Overview

The `plays/` directory contains expansion plays and sub-vertical packs. All files use a flat structure with YAML frontmatter tagging — no subdirectories. Relevance is determined by tag matching, not folder location.

**Current plays:**

| File | Type | Industries | Purpose |
|---|---|---|---|
| `expansion-data-analytics-2026.md` | play | all | Data and analytics expansion motion, Agentforce angle, signal indicators for Researcher |
| `financial-planning-wealth-management.md` | sub-vertical-pack | FSI, Fintech | Industry context, common language, personas, known pressure points for the Financial Planning and Wealth Management segment |

**Planned packs:** Healthcare & Life Sciences, Retail, Manufacturing, CPG, Technology.

Each play file is approximately 80–150 lines. Content is proprietary and not included in this preview.

The play selection logic and frontmatter schema are documented in `memory/README.md`.
